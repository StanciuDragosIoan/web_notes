FlexBox (Flexible Box Module)

Flexbox is a CSS3 layout model that provides an easy way to arrange flex items (child elements) within a container (parent element).It is responsive and mobile friendly, it has no floats, positioning child elements is much easier.

Flex container's margins do not collapse with the margins of its contents. The order of elements can easely be changed without editing the source HTML.

Flexible Box Model Concepts

Flexbox gives us the ability to alter an item width and height to best fit its container's available free space. It is direction agnostic and it is built for small-scale layouts while the "Grid" specification is for large scale layouts.

FlexBox layout model

We have a main axis (the horizontal one) and a cross axis (the vertical axis).The layout is divided between a container and its child elements. The display property is set on the container.

Some flexbox properties 

      /*on container*/
      display: flex|inline flex
      flex-direction: row|column
      flex-wrap: wrap|nowrap|wrapreverse
      flex-basis: /*this property is like the width*/


      /*align items horizontally to the left/right or center*/
      justify-content: fkex-start|flex-end|center 

      /*change alignment for individual flex-items*/
      align-self: flex-start|flex-end|center; 


      /*how the flex items are laid out on the cross axis*/
      align-items: flex-start|flex-end|center; 
                      
      /*aligns content on the cross axis*/ 
      align-content: flex-start|flex-end|center;                                       
      
      flex-grow: /*number for how much an element grows*/
      flex-shrink: /*number for how much an element shrinks*/
