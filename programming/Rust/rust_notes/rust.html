Rust Programming Language

Rust is a multi-paradigm programming language focused on performance and safety, especially safe concurrency. Rust is syntactically similar to C++, but provides memory safety without using garbage collection.

Hello World program in Rust

We start by creating a project directory:

      mkdir ~/projects
      cd ~/projects
      mkdir hello_world
      cd hello_world

Next, in the hello_world directory create a file and name it main.rs

in the main.rs file type:

      fn main() {
        println!("Hello, world!");
      }

Next go back to the terminal and run:

      $ rustc main.rs //compiles the rust code
      $ ./main     //runs it

The output for that will be:

      Hello, world!
